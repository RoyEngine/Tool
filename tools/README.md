## 使用指南

### 选择合适的工具

根据您的需求选择合适的工具：

1. **反编译JAR文件**：
   - 优先使用 **procyon-decompiler-0.6.0**，生成的代码更易读
   - 对于最新Java版本，使用 **cfr-0.152**

2. **代码搜索和转换**：
   - 使用 **ast-grep**，支持多种编程语言

3. **JSON数据处理**：
   - 使用 **jq**，轻量级且功能强大

4. **生成覆盖率报告**：
   - 使用 **coverage_report**，基于Python Coverage库

### 基本使用流程

1. 进入工具所在目录：
   ```bash
   cd tools/工具名称/
   ```

2. 查看工具详细说明：
   ```bash
   cat README.md
   ```

3. 按照README.md中的说明使用工具
